Fonctionnalités

Ce que Sentinel fait au quotidien.

Cette page résume les usages du bot. Pour les noms exacts des commandes, va sur la page Commandes.

Service

Le pointage devient un réflexe simple.

Un membre clique, Sentinel ajoute le rôle, lance le chrono, calcule la durée et met à jour son total quand il termine.

  • Rôle synchronisé
  • Durée calculée
  • Logs de session
  • Historique personnel

Suivi staff

Le staff voit l’activité sans fouiller les salons.

Les agents actifs, les classements et les dernières sessions aident à voir qui travaille et à repérer les oublis.

  • Agents actifs
  • Classements lisibles
  • Historique récent
  • Reset individuel par ID

Modération

Le bot refuse ce qu’il ne peut pas faire.

Sentinel vérifie les rôles, respecte la hiérarchie Discord et note les sanctions utiles au staff.

  • Accès par rôles
  • Bans possibles par ID
  • Historique des sanctions
  • Messages de retour clairs

Annonces

Les annonces restent propres.

Le staff peut publier et modifier des embeds Sentinel pour les annonces, les règles, les recrutements ou les événements.

  • Embeds sous identité Sentinel
  • Modifications libres
  • Quota gratuit lisible
  • Organisation par salon

Dashboard

Le web sert à gérer, pas à se perdre.

Le dashboard regroupe la configuration, les services, les annonces, la modération et l’historique dans des onglets séparés.

  • Assistant de configuration
  • Actions depuis le site
  • Historique des changements
  • Statut serveur
