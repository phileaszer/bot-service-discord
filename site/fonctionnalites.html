Fonctionnalités

Ce que Sentinel automatise vraiment.

Ici, on parle du comportement du bot au quotidien. Les commandes sont listées sur la page Commandes, et le démarrage guidé se trouve sur Installation.

Cycle de service

Le pointage devient un réflexe simple.

Un membre clique, Sentinel applique le rôle, lance ou arrête le chrono, calcule la durée et met à jour son total.

  • Rôle synchronisé
  • Durée calculée
  • Logs de session
  • Historique personnel

Lecture staff

Le staff voit l’activité sans fouiller les salons.

Les agents actifs, les classements et les sessions récentes sont regroupés pour suivre qui travaille et repérer les oublis.

  • Agents actifs
  • Classements lisibles
  • Historique récent
  • Reset individuel par ID

Modération

Les actions sensibles restent encadrées.

Sentinel vérifie l’accès, respecte la hiérarchie Discord et garde une trace exploitable des sanctions.

  • Accès par rôles
  • Bans possibles par ID
  • Journal des sanctions
  • Messages de retour clairs

Annonces

Les messages importants gardent une identité propre.

Le staff peut publier et modifier des embeds Sentinel pour les annonces, les règles, les recrutements ou les événements.

  • Embeds sous identité Sentinel
  • Modifications libres
  • Quota gratuit lisible
  • Organisation par salon

Dashboard

Le web sert à piloter, pas à répéter la documentation.

Le dashboard rassemble la configuration, les actions staff, les statistiques et le journal d’audit dans des onglets séparés.

  • Assistant de configuration
  • Actions immédiates
  • Journal d’audit
  • Statut serveur
