Demarrage rapide

Configurer Sentinel dans Discord

  1. Invite SentinelAjoute le bot sur ton Discord avec le bouton ci-dessous.
  2. Choisis la langueTape /config-langue, puis selectionne Francais ou English.
  3. Choisis le role de serviceTape /config-role et choisis le role qui sera donne aux agents en service.
  4. Choisis le salon de logsTape /config-logs avec l'ID du salon de logs.
  5. Ajoute les roles autorisesTape /config-permissions pour definir qui peut gerer Sentinel.
  6. Publie le panneauTape !service-panel dans le salon ou les membres doivent pointer.
