Démarrage rapide

Configurer Sentinel dans Discord

  1. Invite SentinelAjoute le bot sur ton Discord avec le bouton ci-dessous.
  2. Choisis la langueTape /config-langue, puis sélectionne Français ou English.
  3. Choisis le rôle de serviceTape /config-role et choisis le rôle qui sera donné aux agents en service.
  4. Choisis le salon de logsTape /config-logs avec l'ID du salon de logs.
  5. Ajoute les rôles autorisésTape /config-permissions pour définir qui peut gérer Sentinel.
  6. Publie le panneauTape !service-panel dans le salon où les membres doivent pointer.
