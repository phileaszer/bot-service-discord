Installation

Installer Sentinel sans se perdre.

Le plus important : Sentinel doit être ajouté comme bot Discord, pas seulement comme application de commandes.

Inviter Sentinel Voir le dashboard Aide installation

Premiers pas

Quatre étapes, puis le serveur peut commencer.

Fais ces actions dans l’ordre. Si une étape bloque, le dépannage est juste en dessous.

  1. Invite SentinelUtilise le bouton officiel de cette page pour l’ajouter au bon serveur.
  2. Choisis la langueAvec /config-langue, choisis Français ou English pour ce serveur.
  3. Règle le rôle et les logsUtilise /config-role, /config-logs et /config-permissions.
  4. Publie le bouton de serviceEnvoie !service-panel dans le salon où les membres doivent pointer.

Dépannage / Troubleshooting

Si quelque chose ne fonctionne pas.

Ces vérifications règlent la plupart des soucis d’invitation, de rôles ou de commandes invisibles.

01

Vérifier les droits Discord

La personne qui invite Sentinel doit avoir Gérer le serveur ou Administrateur sur le serveur choisi.

EN: the installer needs Manage Server or Administrator permission on the selected guild.

02

Choisir le bon serveur

Si Discord ouvre directement un serveur qui n'est pas le bon, ferme la fenêtre et relance le lien officiel depuis cette page, sans lien personnalisé.

EN: if Discord locks the wrong guild, restart from the official invite button on this page.

03

Vérifier que Sentinel est bien un bot

Dans les intégrations Discord, Sentinel doit apparaître avec le badge Bot. Si tu vois seulement Commandes, retire l’intégration et réinvite-le.

EN: Sentinel must show as a Bot integration, not only as slash commands.

04

Tester après l’installation

Une fois Sentinel arrivé, utilise /ping puis /diagnostic pour vérifier les rôles et les permissions.

EN: after inviting, run /ping and /diagnostic to confirm that roles and permissions are ready.

Important

Le lien officiel contient deux autorisations.

Sentinel a besoin de bot et applications.commands. Sans les deux, les commandes peuvent apparaître sans que le bot soit vraiment présent sur le serveur.
