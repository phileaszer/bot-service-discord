Commandes

Les commandes Sentinel, classées par usage.

La version gratuite garde l’essentiel. Le Premium ajoute les contrôles avancés utiles aux grandes communautés.

Service gratuit

  • /mes-heuresVoir ses heures
  • /historique-serviceVoir ses 5 dernières sessions
  • /en-serviceVoir les agents actifs
  • /top-serviceVoir le top 10
  • /reset-heuresRéinitialiser une personne, même par ID

Configuration

  • /config-langueChoisir FR ou EN
  • /config-roleDéfinir le rôle de service
  • /config-logsDéfinir le salon de logs par ID
  • /config-permissionsDéfinir les rôles autorisés
  • /config-voirVérifier la configuration
  • /aideAfficher le tutoriel dans Discord

Modération gratuite

  • /avertirAjouter un avertissement
  • /timeoutTimeout temporaire
  • /fin-timeoutRetirer un timeout
  • /expulserExpulser un membre présent
  • /bannirBannir par membre ou ID
  • /purgeSupprimer des messages récents
  • /sanctionsVoir les 10 derniers cas
  • /embedCréer, modifier ou supprimer une annonce Sentinel

Premium préparé

  • /heuresConsulter les heures d’un autre membre
  • /top-semaineClassement de la semaine
  • /resume-serviceRésumé complet du serveur
  • /reset-heures-allReset global
  • /casDétail d’un cas
  • /profil-modProfil de modération complet
  • /modifier-casCorriger un dossier
  • /supprimer-casSupprimer un dossier
  • /unwarnRetirer un avertissement précis
  • /tempbanBan temporaire persistant
  • /unbanDébannir par ID
  • /lockVerrouiller un salon
  • /unlockDéverrouiller un salon
  • /slowmodeMode lent
  • À venirSanctions automatiques après X avertissements
  • /embed creerCréation illimitée d’annonces Sentinel
