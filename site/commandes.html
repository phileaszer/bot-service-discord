Commandes

Les commandes Sentinel, classees par usage.

Le gratuit reste volontairement lisible. Le premium ajoute les controles avances utiles aux grandes communautes.

Service gratuit

  • /mes-heuresVoir ses heures
  • /historique-serviceVoir ses 5 dernieres sessions
  • /en-serviceVoir les agents actifs
  • /top-serviceVoir le top 10
  • /reset-heuresReset une personne, meme par ID

Configuration

  • /config-langueChoisir FR ou EN
  • /config-roleDefinir le role de service
  • /config-logsDefinir le salon de logs par ID
  • /config-permissionsDefinir les roles autorises
  • /config-voirVerifier la configuration

Moderation gratuite

  • /avertirAjouter un avertissement
  • /timeoutTimeout temporaire
  • /fin-timeoutRetirer un timeout
  • /expulserExpulser un membre present
  • /bannirBannir par membre ou ID
  • /purgeSupprimer des messages recents
  • /sanctionsVoir les 10 derniers cas
  • /embedCreer, modifier ou supprimer une annonce Sentinel

Premium prepare

  • /heuresConsulter un autre membre
  • /top-semaineClassement semaine
  • /resume-serviceResume complet serveur
  • /reset-heures-allReset global
  • /casDetail d'un cas
  • /profil-modProfil moderation complet
  • /tempbanBan temporaire persistant
  • /unbanDeban par ID
  • /lockVerrouiller un salon
  • /slowmodeMode lent
  • /embed creerCreation illimitee d'annonces Sentinel
