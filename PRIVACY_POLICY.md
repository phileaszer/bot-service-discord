# Politique de Confidentialite

Derniere mise a jour : 5 juin 2026

Cette Politique de Confidentialite explique quelles donnees Sentinel collecte, pourquoi elles sont utilisees et comment elles sont conservees.

## Donnees collectees

Sentinel collecte uniquement les donnees necessaires a son fonctionnement, notamment :

- identifiant Discord de l'utilisateur ;
- identifiant Discord du serveur ;
- temps de service enregistres ;
- sessions de service ;
- sanctions de moderation enregistrees par le bot, incluant l'action, la raison, la duree le cas echeant, l'utilisateur concerne et le moderateur ;
- configuration du serveur ;
- identifiants des roles utilises pour la gestion du bot ;
- identifiant du salon de logs configure, le cas echeant.

Le bot ne collecte pas le contenu prive des messages, les adresses e-mail, les mots de passe, les informations de paiement ou les conversations privees.

## Utilisation des donnees

Les donnees sont utilisees exclusivement pour :

- suivre les heures de service ;
- afficher les heures personnelles ou celles des membres ;
- afficher l'historique des sessions ;
- generer des classements ;
- afficher les agents actuellement en service ;
- enregistrer et consulter les sanctions de moderation ;
- configurer le role de service, les logs et les roles autorises ;
- verifier le bon fonctionnement du bot.

## Partage des donnees

Les donnees ne sont pas vendues.

Les donnees ne sont pas partagees avec des tiers, sauf si cela est necessaire au fonctionnement technique du bot, a l'hebergement du service ou au respect d'une obligation legale.

## Conservation des donnees

Les donnees sont conservees tant que le bot est utilise sur le serveur concerne ou tant qu'elles sont necessaires au fonctionnement du service.

Les personnes disposant des roles autorises sur un serveur peuvent reinitialiser certaines donnees de service avec les commandes prevues par le bot.

## Suppression des donnees

Pour demander la suppression de donnees associees a un serveur, contactez le proprietaire du projet via le depot GitHub du bot.

Une demande peut necessiter de verifier que la personne qui la formule est autorisee a agir pour le serveur concerne.

## Securite

Les donnees sont stockees dans une base SQLite utilisee par le bot. Des mesures raisonnables sont prises pour limiter l'acces aux donnees aux besoins techniques du service.

Aucun systeme ne pouvant etre garanti comme totalement securise, il est recommande de ne pas utiliser le bot pour stocker des informations sensibles.

## Modifications

Cette politique peut etre modifiee a tout moment. La version publiee sur ce depot fait foi.

## Contact

Pour toute question concernant cette Politique de Confidentialite, contactez le proprietaire du projet via le depot GitHub du bot.
